## Why Eleventy?

Eleventy (11ty) is gaining popularity as a static site generator that prioritizes flexibility and simplicity. Unlike some other SSGs that force you into specific patterns, Eleventy gives you the freedom to structure your project however you want.

## Key Features

- **Zero Config**: Works out of the box with sensible defaults
- **Multiple Template Languages**: Use Nunjucks, Liquid, Markdown, and more
- **Fast Builds**: Incredibly fast compilation times
- **No Client-Side JavaScript Required**: Pure static HTML output

## Perfect Use Cases

Eleventy shines for:
- Personal blogs and portfolios
- Documentation sites
- Marketing websites
- Any project where you want full control over the output

## Getting Started

```bash
npm install -g @11ty/eleventy
eleventy --serve
```

That's it! You're ready to build fast, modern websites with complete control over your tech stack.
